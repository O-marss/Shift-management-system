# 📊 Remote Attendance & Shift Management System

A lightweight, real-time attendance and shift management system designed specifically for **Your Store's sales team**.  
The system streamlines daily operations by tracking attendance, managing shift handovers, and organizing tasks — all in one place.

---

## 🚀 Features

### 👤 Seller Panel

- Secure login using **unique seller code + password**
- Clock **in/out** for attendance tracking
- Submit **shift handovers**
  - Confirm previous shift status
  - Flag issues if any
- View **assigned tasks**
- Access only **personal data** (data isolation per seller)

---

### 🛠️ Admin Dashboard

- Real-time overview of:
  - Attendance records
  - Shift handovers
  - Task completion status
- Manage sellers:
  - Add / remove sellers
- Task management:
  - Assign tasks using seller codes
  - Edit or delete tasks
- Export attendance reports as **CSV**

---

## ⚙️ Tech Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Backend / Database:** Supabase (PostgreSQL + Realtime)
- **Deployment:** Vercel

---

## 📡 Realtime Capabilities

Powered by **Supabase Realtime**, the system ensures:

- Instant updates on attendance logs
- Live task status tracking
- Immediate visibility for admin without refresh
